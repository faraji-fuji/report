Why These Technologies

Semantic HTML improves accessibility and interoperability (content is available to assistive technologies for visitors with disabilities, and to browsers on desktop, mobile, tablet, and other devices alike), improved search engine optimization, lighter code and faster pages, easier code maintenance and styling. Flex box and Grid are layout modules used to create layouts that are more advanced than previous techniques.

Introduction

According to freeCodeCamp, Semantic HTML elements are those that clearly describe their meaning in a human and machine readable way.

Getting Started

To get started, you should be familiar with HTML5 and CSS3. Semantic HTML elements are introduced in HTML5 and include elements such as article, aside header and footer. CSS3 covers the contemporary layout modules such as Flex Box and The Grid System.

The Grid System

Grids are broken down into evenly spaced colums and rows. The columns and rows are used as guides for laying out page components. Gutters are included to add consistent spacing between columns. Page components are arranged within the columns. Some grid terminologies include grid container and grid items. Grid container is the parent element while grid items are the child items within the grid container.

Flex Box

In flex box layout module, items are aligned on a single axis. It is great for distribution of components across a single axis. Some common flex box terminologies include; flec container and flex item. Flex container is the parent element while flex item is the child element within the flex container. Flex items are laid out along the main axis. The direction is horizontal by default meaning flex items are arranged in rows. The direction of the main axis can be changed to run vertically.
